Zindex_Label -- 获取对象索引

类型

ZVOBJECT

描述

获取ZVOBJECT对象索引，对象索引为一个数字编号，类似于指向数据的指针

语法

index = Zindex_Label(obj)

 

参数：

obj：ZVOBJECT定义的对象参数

适用控制器

支持ZV功能或者5系列以上的控制器

例子

例一：

ZVOBJECT img

LOCAL index

ZV_ImgRead(img,"logo.png",0) '读取一张图像

index = ZINDEX_LABEL(img) '获取图像对象的索引

 

例二：

ZVOBJECT ZvArray1(100)

LOCAL ZvIndex1

ZvIndex1 = ZINDEX_LABEL(ZvArray1)'建立obj数组索引

ZV_ImgRead(ZINDEX_ZVOBJ(ZvIndex1)(1),"logo.png",0) '读取图像

ZV_LATCH(ZINDEX_ZVOBJ(ZvIndex1)(1),0) '显示图像

 

例三：

ZVOBJECT ZvArray2(100)

LOCAL ZvIndex2

ZvIndex2 = ZINDEX_LABEL(ZvArray2)

ZV_ImgRead(ZINDEX_ZVOBJ(ZvIndex2)(1),"0.bmp",0)

ZV_ImgRead(ZINDEX_ZVOBJ(ZvIndex2)(10),"1.bmp",0)

SHOW_IMG(ZINDEX_ZVOBJ(ZvIndex2))

END

 

GLOBAL SUB SHOW_IMG(BYREF ObjArr(100) as ZVOBJECT)

    ZV_LATCH(ObjArr(1),0) '显示图像

    DELAY(2000)

    ZV_LATCH(ObjArr(10),0) '显示图像

    DELAY(2000)

    ZV_LATCH(ObjArr(1),0) '显示图像

ENDSUB
