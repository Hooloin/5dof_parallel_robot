ZG_READLINE -- 读取加工行代码

类型

辅助功能

描述

读取指定行内容（行号），返回行字符串

运行文件时也可使用该指令，可以追踪到运行的子程序代码

语法

string = ZG_READLINE (linenum)

linenum：行号，0-为当前运行行/编辑行

string：返回字符串

 

ZG_READLINE (linenum, tabfirst)

linenum：行号

tabfirst：存储行字符串的起始table位置

适用控制器

通用

例子

dim string(256)

string = ZG_READLINE (1)

trace string    ’打印第1行代码

 

ZG_READLINE (1, 1000)

trace tablestring(1000, 128)    ’打印第1行代码，最长128个字符
