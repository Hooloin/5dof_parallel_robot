DRIVE_TORQUEMIN -- 驱动器最小力矩限制

类型

EtherCAT轴状态

描述

驱动器的当前力矩的最小限制。

必须设置正确的ATYPE（设置为65/66/67）与DRIVE_PROFILE以后才能读取。

语法

可读：var = DRIVE_TORQUEMIN(axis)

可写：DRIVE_TORQUEMIN(axis) = value

axis：轴号

适用控制器

带EtherCAT接口或RTEX接口

例子

SLOT_SCAN(0)

IF NODE_COUNT(0)>0 THEN

ATYPE(0)=65                 '总线位置控制

DRIVE_PROFILE(0)=1         'pdo设为1，含有返回力矩功能

? DRIVE_TORQUEMIN (0)         '打印轴0的力矩

ENDIF

相关指令

ATYPE，DRIVE_PROFILE，DRIVE_TORQUE，DRIVE_TORQUEMAX
