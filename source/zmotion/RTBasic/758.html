ZV_SysRunErr -- 运行错误码

类型

错误处理

描述

获取或修改运行错误码，记录的为第一个运行出错的指令的错误码。参数指定要获取错误信息的任务号，可以不带参数则为当前任务，需注意RTSys命令行和HMI界面都有独立的任务号

注意：记录的是运行任务第一个报错的视觉指令的错误码，未清零情况下将一直保持不会改变。

 

别名：ZV_RUNERR、ZV_SysErr

语法

获取任务运行错误码：error = ZV_SysRunErr(taskid)

修改任务运行错误码：ZV_SysRunErr(taskid) = 0

适用控制器

支持ZV功能或者5系列以上的控制器

例子

举例1：

IF(0 <> ZV_SysRunErr)THEN '当前任务错误码不为0则返回

    RETURN

ENDIF

 

举例2：

ZV_SysRunErr(taskid) = 0 '清空任务taskid的运行错误
