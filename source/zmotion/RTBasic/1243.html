ZV_DraColor -- 生成颜色

类型

绘制

描述

使用r、g、b，生成颜色值

别名：ZV_COLOR

语法

value = ZV_DraColor(r,g,b)

 

参数：

r：红色分量的值，范围[0,255]

g：绿色分量的值，范围[0,255]

b：蓝色分量的值，范围[0,255]

返回值：颜色值

适用控制器

支持ZV功能或者5系列以上的控制器

例子

' 生成图像用于绘制

ZVOBJECT img

TABLE(0,0,0,0)

ZV_ImgGenConst(img,640,480,3,0,0) '生成图像

 

LOCAL color

color = ZV_DraColor(255,0,0) '红色

ZV_DraLine(img,50,50,600,50,color) '绘制直线

ZV_DraLine(img,50,100,600,100,ZV_DraColor(0,255,0)) '绘制绿色直线
