ZG_RUNMODE -- NC文件启动模式

类型

程序控制

描述

设置/获取NC文件启动模式

语法

var = ZG_RUNMODE ([,channel])  ’获取当前启动模式

 

ZG_RUNMODE ( mode [,channel])  ’设置文件启动模式

        mode：启动模式

mode

内容说明

bit 0 = 0

正常启动-单次加工

bit 0 = 1

循环加工，遇M30会跳转到首行

bit 1 = 1 (2)

模拟加工，最终只计算生成轨迹点，可通过绘图指令显示

开启模拟加工时，循环加工将失效

bit 2 = 1 (4)

单段运行模式，执行单段后立即停止，通过ZG_PROGSTEP指令去执行下一段

bit3(8)

语法检查，加工前先进行语法检查

适用控制器

通用

例子

ZG_RUNMODE(4+2)    ’设置单段模拟加工
