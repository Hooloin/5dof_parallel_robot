HMI_XYChartRange -- XY曲线图上下限范围

类型

操作指令

描述

设置曲线图上下限范围

语法

HMI_XYChartRange(winid, controlid, channel [, minx, miny, maxx, maxy])

winid：HMI文件里面窗口编号

controlid：元件编号

channel：指定通道，编号从0开始

minx, miny：下限范围

maxx, maxy：上限范围

适用控制器

支持RTHmi

例子

'通道0的显示范围设置为(0,0)->(100,100)

HMI_XYChartRange(10, 1, 0, 0, 0, 100, 100)
