U_READ -- 从U盘读取

类型

存储指令

描述

从外部存储器读取数据到变量或数组里面。

 

不具备外部存储接口的控制器不支持此命令。

文件格式为32位ieee浮点数顺序存储，一个变量或一个数组元素占用一个浮点数，可以通过PC事先做好文件，然后用U_READ指令读取。

U盘读取文件系统支持fat32和fat16，不支持ntfs。

4系列以下建议使用USB2.0的U盘，4系列或以上支持USB3.0。

语法

U_READ sect_num, [,varname] [,arrayname] [,arrayname(a)][, arrayname(a,length)]

sect_num：文件编号，对应到SD【filenum】.BIN

varname：变量名

arrayname：数组名，可以为TABLE，VR

a：操作的数组索引

length：操作的数组元素个数

适用控制器

带U盘接口

例子

IF  U_STATE = TRUE THEN   'U盘已插入

        U_READ  1, VAR, TABLE(0), ARRAY2(1)  '读取U盘文件SD1数据

ENDIF

相关指令

U_WRITE，U_READ2，U_STATE
