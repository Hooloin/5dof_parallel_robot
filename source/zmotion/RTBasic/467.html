ZAR_CONTROL -- 查看控制器类型

类型

U盘函数

描述

查看主控制器U盘的ZAR程序文件的对应控制器类型，此类型在ZDevelop中设置，避免主控制器和从控制器的文件混淆。

语法

id = ZAR_CONTROL ("ufilename")

ufilename        ：U盘文件名，ZAR文件

适用控制器

4系列以上控制器支持，支持版本号20170618

例子

id = ZAR_CONTROL("1.ZAR")

IF(id/3000=3)  Then   'ZHD系列示教盒

PRINT  "zhd program"

ENDIF

相关指令

SEND_ZAR，CONTROL
