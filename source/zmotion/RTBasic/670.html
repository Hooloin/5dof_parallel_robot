DrawEx_TabVect -- 从table中绘图（批量绘图）

类型

绘图指令

描述

在指定区域绘table图形，图形数据按一定格式存储在指定table位置中

此指令只能在自定义元件的绘图函数内使用。

语法

DrawEx_TabVect(tablein, offx, offy, scalex, scaley)

tablein：输入table地址

offx, offy：显示图形偏移

（>0向左/上偏移，<0向右/下偏移，=0不偏移）

scalex：x方向显示缩放比例，1.0不缩放

scaley：y方向显示缩放比例，1.0不缩放

适用控制器

支持RTHMI

例子

'封装table图形数据，将需要画的图形按指定格式填入table中

local iTableStart, iTableCur

iTableStart = 20000  '起始table下标

 

'第1项是总大小，最后再填入

iTableCur = iTableStart + 1

 

'第一项，画个填充矩形rect=(20,20,200,200), R=10，5个参数，占用2+5长度

table(iTableCur, 354, 5, 20, 20, 200, 200, 10)

iTableCur = iTableCur + 7

 

'第二项，画个矩形rect=(20,20,200,200), R=10，5个参数，占用2+5长度

table(iTableCur, 304,5, 20, 20, 200, 200, 10)

iTableCur = iTableCur + 7

 

'第三项，画个旋转矩形rect=(220,220,200,200), 旋转弧度=45*PI/180，5个参数，占用2+5长度

table(iTableCur, 305,5, 220, 220, 200, 200, 45*PI/180)

iTableCur = iTableCur + 7

 

'最后补充总大小

table(iTableStart) = iTableCur-iTableStart

 

'画图，将table中存储的图形显示出来

DrawEx_TabVect(iTableStart , 0, 0, 1.0, 1.0)
