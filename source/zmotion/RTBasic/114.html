DMSEARCH -- 数组搜索

类型

语法指令

描述

根据元素的值，搜索该元素在数组中的位置，返回找到的第一个数组索引，搜索不到返回-1。

 

谨慎对超长数组进行操作，特别是数组TABLE。

语法

Pos = DMSEARCH( array, startpos, offset, maxtimes, value)

array 数组名

startpos：搜索起始位置

offset：每次搜索跳过的间隔

maxtimes：最多判断的次数

value：搜索的数值

返回：

Pos：数组的索引，-1没有找到

适用控制器

通用

例子

DIM ruturn, value

DIM arr1(10)

FOR i = 0 TO 9

        arr1(i) = i

NEXT

value = DMSEARCH(arr1,0,1,10,3)

ruturn = DMSEARCH(arr1,0,1,10,20)

IF value = 3 AND ruturn = -1 THEN

    ?"查找成功"

ELSE

    ?"查找失败"

END IF
