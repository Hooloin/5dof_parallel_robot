ZV_ObjIsInit -- 是否初始化

类型

ZVOBJECT通用操作

描述

判断ZVOBJECT变量是否初始化。

未初始化：指变量未使用过或执行过解引用；

已初始化：变量可以通过ZV_ObjDetach解引用恢复未初始化状态。

支持在线命令，使用无需传入TABLE索引参数

注意：已初始化的变量可能为空，ZV_ObjClear不影响初始化状态

语法

ZV_ObjIsInit(obj,tabId) 或 value = ZV_ObjIsInit(obj)

 

参数：

obj：ZVOBJECT定义的参数

tabId：TABLE索引，输出结果，是否初始化，1-已初始化、0-未初始化

适用控制器

支持ZV功能或者5系列以上的控制器

例子

ZVOBJECT  obj

ZV_ObjIsInit(obj,0)

IF TABLE(0)=1 THEN

PRINT "obj变量已初始化"

ENDIF

相关指令

ZV_ObjDetach
