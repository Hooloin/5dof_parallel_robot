ZG_SPECIAL -- 特殊功能转化坐标

类型

坐标转换

描述

将要移动的坐标位置，做特殊功能处理，转化为特殊处理后的坐标，若当前未开启特殊功能，则不会进行转换处理。

只能输入程序坐标，机械坐标不参与特殊计算

G代码的特殊功能包括：镜像功能、比例缩放功能、坐标旋转功能，还有刀补功能（刀补处理比较特殊，该指令的转化不包括刀补功能）

语法

ZG_ SPECIAL(first, ifrecord, ifRel)

        first：table起始索引，第一个为轴数，第二个开始为轴坐标

        ifrecord：是否记录该坐标为最新程序坐标？刀补计算时需要

        ifRel：是否相对模式

 

var = ZG_ SPECIAL()

        返回当前是否开启特殊功能

        bit0：是否开启坐标旋转功能G68（1）

        bit1：是否开启比例缩放功能G51（2）

        bit2：是否开启镜像功能G51.1（4）

        bit3：圆弧方向是否反向（8）

适用控制器

通用

例子

dim tabfirst = 1000

 

table(tabfirst) = 3

table(tabfirst+1) = 100

table(tabfirst+2) = 100

table(tabfirst+3) = 100

’ 输入程序坐标，相对模式，输出机械坐标

ZG_ SPECIAL(tabfirst, 0, 1)   

MOVE(table(tabfirst+1), table(tabfirst+2), table(tabfirst+3))

 

table(tabfirst+1) = 200

table(tabfirst+2) = 200

table(tabfirst+3) = 200

输入程序坐标，绝对模式，输出机械坐标

ZG_ SPECIAL(tabfirst, 0, 0 )   ’

MOVEABS(table(tabfirst+1), table(tabfirst+2), table(tabfirst+3))

 

PS：对客户来说，一个简单的MOVE需要这么多行代码，比较麻烦，可以选择封装成sub函数。

sub sub_move(x, y, z)

        table(tabfirst) = 3

        table(tabfirst+1) = x

        table(tabfirst+2) = y

        table(tabfirst+3) = z

        ’ 输入程序坐标，相对模式，输出机械坐标

        ZG_ SPECIAL( tabfirst, 0, 1 )   

        MOVE(table(tabfirst+1), table(tabfirst+2), table(tabfirst+3))

end sub
