ZV_ObjIsEmpty -- 是否为空

类型

ZVOBJECT通用操作

描述

判断ZVOBJECT变量是否为空，为空指变量无类型（如新定义变量）或变量数据区为空（如调用ZV_CLEAR）。

支持在线命令，使用无需传入TABLE索引参数

 

别名：ZV_ISEMPTY

语法

ZV_ObjIsEmpty(obj,tabId) 或 value = ZV_ObjIsEmpty(obj)

 

参数：

obj：ZVOBJECT定义的参数

tabId：TABLE索引，输出结果，是否为空，1为空、0非空

适用控制器

支持ZV功能或者5系列以上的控制器

例子

ZVOBJECT objtest

ZV_ObjIsEmpty(objtest,0)

IF (TABLE(0)=1) THEN

    PRINT "obj变量为空"

ELSE

    PRINT "obj变量不为空"

ENDIF
